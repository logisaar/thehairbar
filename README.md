# The Hair Bar

Premium Hair Salon in Chandak.

## Project info

A modern web application built for The Hair Bar, featuring a premium design and seamless user experience.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the application for production:

```sh
npm run build
```
